### 所需参数

- mchId(商户号)：
> 账户中心 - 商户信息 - 商户号

- 商户类型：
> 账户中心 - 商户信息 - 商户类型

- appId(绑定的公众号appId)：
> 产品中心 - AppID账号管理 - appId

- apiV3Key(APIv3秘钥)：
> 账户中心 - API安全 - 设置APIv3秘钥

- API证书(apiclient_key.pem)：
> 账户中心 - API安全 - 申请API证书 

- certSerialNo(API证书序列号)：
> 账户中心 - API安全 - 申请API证书 - API证书管理 -证书序列号
